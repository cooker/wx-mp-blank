# 微信公众号文章编辑器

一个基于 Vue 3 的微信公众号文章编辑器，支持文字折叠和自动填充功能。

## 功能特性

- ✨ 实时字数统计
- 📝 文字编辑功能
- 🔽 文字隐藏到最小显示（折叠/展开）
- 🔄 字数不足500字时自动填充随机字符串
- 🎨 现代化的UI设计

## 安装依赖

```bash
npm install
```

## 运行项目

```bash
npm run dev
```

项目将在 http://localhost:3000 启动

## 构建项目

```bash
npm run build
```

## 使用说明

1. **编辑文章**：在文本框中输入或粘贴文章内容
2. **查看字数**：顶部实时显示当前字数，不足500字时会高亮提示
3. **自动填充**：当字数不足500字时，点击"自动填充至500字"按钮会自动填充随机文字
4. **折叠编辑器**：点击"隐藏到最小"按钮可以将编辑器折叠到最小显示，再次点击可展开

## 技术栈

- Vue 3
- Vite
- CSS3
